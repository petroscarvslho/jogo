# 06 - Diluições de Bancada

## Lidocaína 1% (a partir de 2%)
- 5 mL lidocaína 2% + 5 mL SF = 10 mL de lidocaína 1%.

## Bupi/levobupi 0,25% (a partir de 0,5%)
- 5 mL de 0,5% + 5 mL SF = 10 mL de 0,25%.

## Botão anestésico menos doloroso
- 1 mL lidocaína 1% + 0,1 mL bicarbonato 8,4% (1:10).

## Dextrose 5% (a partir de 25%)
- 1 mL dextrose 25% + 4 mL SF = 5 mL dextrose 5%.

## Segurança de preparo
- Identificar seringa com fármaco/concentração.
- Conferência cruzada antes de injetar.
